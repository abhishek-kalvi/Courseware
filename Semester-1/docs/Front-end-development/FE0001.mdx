### Installing VS Code

VS Code includes great tooling for web technologies such as JSX/React, HTML, CSS, SCSS, Less, and JSON. Throughout the course, we'll be using the VS code extensively. It is not just an editor, the features supported by VS Code reduce the tasks of any developer significantly. Along with VS code, we will be installing a few additional extensions too.

System requirements for the VS Code:

Visual Studio Code is a small download (&lt; 100 MB) and has a disk footprint of 200 MB. VS Code is lightweight and should easily run on today's hardware.

**How to install VS Code?**

Open VS Code in your browser using this link -  [Download Visual Studio Code](https://code.visualstudio.com)

Now VS code would have detected your OS.

Download and install VS Code for your system (follow the instructions for installation)

After installing VS Code editor, you need to install some basic plugins to make the platform a better place to code. So, install these as well:

ESLINT - [Click here to install it](https://marketplace.visualstudio.com/items?itemName=dbaeumer.vscode-eslint)

BEAUTIFY - [Click here to install it](https://marketplace.visualstudio.com/items?itemName=HookyQR.beautify)

PRETTIER - [Click here to install it](https://marketplace.visualstudio.com/items?itemName=esbenp.prettier-vscode)
